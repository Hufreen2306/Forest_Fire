# 🌲 Forest Fire Prediction & Risk Mapping

## 📌 Overview
Forest Fire Prediction & Risk Mapping is a GIS and Machine Learning project designed to predict forest fire risk based on environmental and weather conditions. The system analyzes historical data to identify fire-prone areas and visualizes the results using GIS-based risk maps.

## 🎯 Objectives
- Predict the likelihood of forest fires.
- Identify high-risk areas using GIS.
- Support early warning and disaster management.
- Assist in forest conservation and resource planning.

## ✨ Features
- Forest fire risk prediction using Machine Learning
- GIS-based risk mapping
- Weather and environmental data analysis
- Data visualization through charts and maps
- High-risk zone identification

## 🛠️ Technologies Used
- Python
- QGIS
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Dataset
The dataset includes:
- Temperature
- Humidity
- Wind Speed
- Rainfall
- Latitude
- Longitude
- Fire Occurrence

## ⚙️ Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Prediction
6. GIS Risk Mapping
7. Result Visualization
